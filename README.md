# e-plantShopping

## Paradise Nursery Shopping Application

This project is a React-based shopping application for an online plant nursery.
The application allows users to browse plants, add items to a shopping cart,
and manage their cart using Redux for global state management.

### Technologies Used
- React
- Redux Toolkit
- JavaScript
- HTML
- CSS

This repository is named e-plantShopping and contains the source code
for the Paradise Nursery Shopping Application.
